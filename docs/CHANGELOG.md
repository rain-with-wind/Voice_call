# 更新日志

## [1.0.0] - 2026-06-11

### 新增
- 模块化项目结构 (`voice_call_app/`)
- 中文终端界面（状态栏、提示信息、帮助文档）
- `--list-devices` 列出所有音频设备
- `--input-device` / `--output-device` 手动指定音频设备
- 静音检测提示
- Fernet 端到端加密（基于共享密码）
- SHA-256 密码认证握手
- Windows 终端 UTF-8 自动配置

### 变更
- 默认采样率从 44100Hz 改为 48000Hz
- 默认声道数从 2 改为 1
- 从单文件脚本重构为多模块架构

### 依赖
- `pyaudio==0.2.14` — 音频采集与播放
- `cryptography==41.0.7` — 加密支持（可选）
